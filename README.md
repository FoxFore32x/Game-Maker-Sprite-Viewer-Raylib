# Game Maker 2 Sprite Viewer

Basically this is a reader for Game Maker 2 files.

Features:
- Sprites can load on Real Dreamcast hardware (Under 16MB of RAM)
- BZ2 + QOI Support
- Sprite and Texture Page viewer (If Texture page is >= 2048, crashes on Dreamcast fue to lack of RAM)
- Full Sprite animation
